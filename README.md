## 🌟 Overview

KineticCare is a browser-native rehabilitation platform that uses AI-powered pose estimation and augmented reality visualization to guide patients through recovery exercises in real time.

Using MediaPipe BlazePose, TensorFlow.js, and Three.js, KineticCare performs real-time biomechanical analysis directly in the browser while providing instant posture correction and exercise feedback.

Unlike traditional rehabilitation systems, KineticCare requires no wearable devices, no mobile application downloads, and no specialized hardware—just a webcam and a browser.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Dushyant107/KineticCare.git

cd KineticCare
```

### Backend Setup

```bash
cd server

npm install

npm start
```

### Frontend Setup

```bash
cd client

npm install

npm run dev
```
